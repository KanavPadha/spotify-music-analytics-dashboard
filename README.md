#  Spotify Music Analytics Dashboard

Analyzed Spotify songs to explore how **danceability**, **energy**, and **duration** influence **popularity** using Python, SQL, and Tableau.

## Tools Used
- **Python (Google Colab):** Data cleaning & preprocessing with Pandas and NumPy
- **SQL (SQLite):** Genre-level aggregation
- **Tableau:** Dashboard visualization

## 📊 Key Insights
- Dance & R&B are the most popular genres.
- Higher **danceability** and **energy** correlate with higher popularity.
- Average song duration: **3.8 minutes** (short, upbeat songs perform better).
- Energy and popularity show a **positive correlation**.
